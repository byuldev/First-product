# AI HACKING LAB

20명 내외 동시접속 워크숍을 위한 간단한 웹 시스템입니다.

## 기능
- QR 스캔 기반 참가 등록
- 등록 대기열 인원 표시 및 자동 시작(임계치 도달 시)
- 운영자 페이지에서 임계치 조정/즉시 시작/리셋
- 공격/방어 영역별 3문제씩, 총 21문제 제공

## 실행 (의존성 설치 없이 바로 실행)
```bash
python3 app.py
```

브라우저에서 `http://localhost:5000` 접속.

## 주요 경로
- `/` : 메인 + 등록 QR
- `/register` : 참가자 등록
- `/waiting/<token>` : 대기 화면(자동 폴링)
- `/challenges` : 시작 후 문제 목록
- `/admin` : 운영자 제어판
